# Empirical Verification of Moore's Law

##  Overview
半導体のトランジスタ数が指数関数的に増加するという「ムーアの法則」について、1971年から現在までの主要CPUの歴史的データを用いて実証的に検証しました。G検定の学習中に何となく興味が出て徒然なるままに分析したので、ミスあったら報告お願いします。
複雑な予測モデルを組むのではなく、生データの性質（指数関数的増加）を正確に捉え、最適な可視化手法を選択することに焦点を当てています。

## Analytical Approach & Findings
本分析では、意図的にシンプルな散布図を採用し、**Y軸を対数スケール（Log Scale）に変換**してプロットしています。

* **手法の意図**: 指数関数的な成長を通常の線形スケールで描画すると、近年のデータのみが極端に強調され、過去のトレンドとの連続性が視認できません。対数スケールを用いることで、理論上の「倍増」が「直線」として表現され、法則の妥当性を視覚的に一目で検証可能になります。
* **結果**: 過去約50年にわたり、トランジスタ数がほぼ一直線（対数グラフ上）に成長していることが確認でき、ムーアの法則の強力な実証データとなっています。
